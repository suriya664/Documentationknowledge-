API Reference

Complete API documentation with examples

Authentication

All API requests require authentication using an API key. Include your key in the request header:

Authorization: Bearer YOUR_API_KEY

Endpoints

GET /api/v1/docs

Retrieve all documentation entries

fetch('/api/v1/docs', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
})
{
  "status": "success",
  "data": [
    {
      "id": 1,
      "title": "Getting Started",
      "content": "..."
    }
  ]
}
POST /api/v1/docs

Create a new documentation entry

PUT /api/v1/docs/:id

Update an existing documentation entry

DELETE /api/v1/docs/:id

Delete a documentation entry

GET /api/v1/docs/search

Search documentation entries by query

GET /api/v1/users/me

Get current user information

PUT /api/v1/users/me

Update current user profile

GET /api/v1/analytics

Retrieve analytics data for documentation
